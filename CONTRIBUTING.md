
# Contributing to PlayVault

PlayVault welcomes contributions from the gaming community.

## How to Contribute

### Report Errors
Found a broken link or outdated guide? Contact us:
- Site: https://game.xxym8.com/contact.html
- Email: hello@xxym8.com

### Suggest Games
Know a game that deserves coverage? Use the contact form at:
https://game.xxym8.com/contact.html

### Submit Data
Have competitive data (win rates, pick rates, DPS numbers)?
Share it via our contact page and we will incorporate verified data into relevant guides.

## What We Look For
- Games with genuine search demand
- Specific, answerable questions (not generic topics)
- Original analysis backed by real data
- No affiliation with game studios
